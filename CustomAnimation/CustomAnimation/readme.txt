
一，以下为鱼类型配置文件说明 （注：统一为int类型值）

1.key 为对应与类型大类
2.value 格式 a~b 标示小类从a到b联系值，如果类型不连续配置为：a~b,c,d~e 解析时将分段值合并处理
3.显示时根据大类，小类初始化界面，图片文件名称格式为 [@"%04d_%03d",type, subType],如：0001_001@.png
4.value中为操作的子类，按照先后顺序显示
5.如果一个大类下子类需要做区分呈现，如工聊，自定义大类格式:大类0x（1901-HR,1902-员工)
6.特殊处理，有的分类中可能会插入其他类型的小类（如乞讨鱼属于冒泡鱼类型，显示时放入杂鱼中），这类处理办法格式 "大类:小类"  : a~b,A:c,...


二，特殊配置（key 2001以上）
1.动画表情 key : 2001 value为可以显示的序号段
2.静态表情 key : 2002 value为可以显示的序号段
3.禅鱼    key :  2003 value为可显示的鱼
